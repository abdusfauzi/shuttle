# 16 - Glossary

- `HostDefinition`: one executable menu leaf containing `name` and `cmd`.
- `GroupNode`: menu section containing child host/group nodes.
- `TerminalTarget`: selected terminal backend (Terminal/iTerm/Warp/Ghostty/Virtual).
- `TerminalWindowMode`: routing mode (`new`, `tab`, `current`, `virtual`).
- `Backend`: terminal-specific command execution implementation.
- `Parity`: behavior match with prior release for same config input.
