# Legacy Script Archive

This directory keeps deprecated terminal scripting artifacts that are no longer part of active runtime or regression gates.

- `warp/`:
  - `Warp-stable-*.applescript`: legacy UI-driven Warp helper AppleScripts.
  - `compile-WarpStable.sh`: legacy helper compiler script.
  - `warp_legacy_compile_check.sh`: legacy compile verification helper.
- `archive/legacy-scripts/warp/Warp-stable-*.scpt`:
  - archived compiled outputs from historical Warp legacy helper scripts.

If you need these assets for historical comparison, they are preserved here only.
